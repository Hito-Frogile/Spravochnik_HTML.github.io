HTML - справочник

Сайт-справочник, созданный с помощью чистого HTML (возможно с минимальным добавлением CSS для стилей), — это простой, эффективный и фундаментальный инструмент.

Что такое HTML и где применяется?

Каждый сайт, размещённый в интернете, написан на языке разметки - HTML. он нужен, чтобы размещать (структурировать) на веб-странице элементы: текст, изображения, таблицы и так далее. он объясняет браузеру, где что находится.

HTML один из фундаментов всех сайтов, который сам по себе не является языком программирования. вместе с ним используют два других языка: CSS и JavaScript.

Популярные текстовые редакторы для работы с HTML

WebStorm

интегрированная среда разработки (IDE) от компании JetBrains, предназначенная для работы с HTML, JavaScript и TypeScript.

Visual Studio Code

текстовый редактор, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб- и облачных приложений.

Sublime Text

кросс-платформенный текстовый редактор, предназначенный для работы с кодом.
